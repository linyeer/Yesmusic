# Yesmusic 🎵

欢迎来到 **Yesmusic**，这是一个基于 **Vue3**，**Vite**，**Element Plus** 和 **Windi CSS** 的前端，以及基于 **Node.js** 的后端的本地在线音乐播放网站。

## 🚀 功能

- **在线音乐播放**：享受无缝的在线音乐体验。
- **本地音乐播放**：上传并播放您自己的音乐文件。
- **用户友好的界面**：使用Vue3，Vite，Element Plus和Windi CSS创建的现代化用户界面。

## 🛠 技术栈

- **前端**：Vue3 + Vite + Element Plus + Windi CSS
- **后端**：Node.js
